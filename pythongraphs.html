Python Graphs

In this project we needed to analyze VO2 max test data
and generate graphs formatted in APA style to visualize the data

I've split the code into 2 sections for each portion of the project. For the first part, we were to modify existing code pre existing code to suit our purposes. The second part involved writing original code to graph different data than the first section.

This section reads in data from a CSV. It modifys several of the data column names to make them more usable or accurate. It then generates and formats a two panel graph, plotting VO2 and VE over time. It also determines and labels VO2max on the graph.
